## About Me

I'm a backend and DevOps engineer focused on building systems that actually work in production — not just in theory.

I design and operate systems where backend logic, infrastructure, messaging, and real-world device behavior need to work together reliably. I think in systems before I think in individual features: how data flows, where responsibilities live, how failures are handled, and how things behave under real load.

Most of my work sits at the boundary between architecture and implementation — turning real operational requirements into clean backend services, telemetry pipelines, and deployment workflows that are practical to run and scale.

- Backend / DevOps engineer with production system experience
- Strong focus on distributed systems and event-driven architectures
- Comfortable across backend, infrastructure, messaging and integration layers
- Experience with telemetry, command flows and cloud-connected systems
- Computer Engineering graduate from Beykent University
- I develop games for fun
